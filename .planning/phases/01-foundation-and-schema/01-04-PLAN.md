**Step 2 — Enums (define before tables that use them):**
```typescript
export const staffRoleEnum = pgEnum("staff_role", ["teacher", "counselor", "principal"]);
export const attendanceStatusEnum = pgEnum("attendance_status", ["present", "absent", "tardy", "excused"]);
export const gradeTypeEnum = pgEnum("grade_type", ["midterm", "final", "quarter", "assignment"]);
export const testTypeEnum = pgEnum("test_type", ["SAT", "PSAT", "ACT", "AP", "other"]);
export const semesterEnum = pgEnum("semester", ["fall", "spring", "full_year"]);
export const insightTypeEnum = pgEnum("insight_type", ["at_risk", "intervention", "roadmap", "trend"]);
```

**Step 3 — Table 1: staff_profiles**
Columns: id (uuid PK — NO defaultRandom), email (text notNull), fullName (text), role (staffRoleEnum notNull), createdAt (timestamp withTimezone defaultNow notNull).

**Step 4 — Table 2: students**
Columns: id (uuid PK defaultRandom), firstName (text notNull), lastName (text notNull), gradeLevel (smallint notNull — grade 9-12), counselorId (uuid references staff_profiles.id nullable), isActive (boolean notNull default true), enrolledAt (date), createdAt (timestamp withTimezone defaultNow notNull).
Indexes: students.gradeLevel, students.counselorId, students.isActive.

**Step 5 — Table 3: classes**
Columns: id (uuid PK defaultRandom), courseName (text notNull), courseCode (text), teacherId (uuid references staff_profiles.id notNull), semester (semesterEnum notNull), schoolYear (text notNull — e.g., "2024-2025"), createdAt (timestamp withTimezone defaultNow notNull).
Indexes: classes.teacherId.

**Step 6 — Table 4: enrollments**
Columns: id (uuid PK defaultRandom), studentId (uuid references students.id onDelete cascade notNull), classId (uuid references classes.id onDelete cascade notNull), enrolledAt (timestamp withTimezone defaultNow notNull).
Indexes (array form): uniqueIndex on (studentId, classId), index on studentId, index on classId.

**Step 7 — Table 5: attendance_records**
Columns: id (uuid PK defaultRandom), studentId (uuid references students.id onDelete cascade notNull), classId (uuid references classes.id nullable — some attendance is school-wide not class-specific), date (date notNull), status (attendanceStatusEnum notNull), notes (text), recordedBy (uuid references staff_profiles.id), createdAt (timestamp withTimezone defaultNow notNull).
Indexes: (studentId, date).

**Step 8 — Table 6: grades**
Columns: id (uuid PK defaultRandom), studentId (uuid references students.id onDelete cascade notNull), classId (uuid references classes.id onDelete cascade notNull), gradeType (gradeTypeEnum notNull), score (numeric("score", { precision: 5, scale: 2 }) — supports 0.00-100.00), letterGrade (text), notes (text), gradedAt (date), createdAt (timestamp withTimezone defaultNow notNull).
Indexes: grades.studentId, (studentId, classId).

**Step 9 — Table 7: standardized_tests**
Columns: id (uuid PK defaultRandom), studentId (uuid references students.id onDelete cascade notNull), testType (testTypeEnum notNull), testDate (date), totalScore (integer), mathScore (integer), readingScore (integer), writingScore (integer), targetScore (integer), notes (text), createdAt (timestamp withTimezone defaultNow notNull).
Indexes: standardizedTests.studentId.

**Step 10 — Table 8: graduation_plans**
Columns: id (uuid PK defaultRandom), studentId (uuid references students.id onDelete cascade notNull UNIQUE — 1:1), creditsEarned (numeric("credits_earned", { precision: 4, scale: 1 }) default "0"), creditsRequired (numeric("credits_required", { precision: 4, scale: 1 }) default "24"), onTrack (boolean default true), planData (jsonb — flexible for credit category breakdown), updatedAt (timestamp withTimezone defaultNow notNull), createdAt (timestamp withTimezone defaultNow notNull).
Note: Use `uniqueIndex` on studentId to enforce 1:1.

**Step 11 — Table 9: college_prep_plans**
Columns: id (uuid PK defaultRandom), studentId (uuid references students.id onDelete cascade notNull UNIQUE — 1:1), targetSchools (jsonb — array of school objects), applicationDeadline (date), essayStatus (text), recommendationStatus (text), notes (text), updatedAt (timestamp withTimezone defaultNow notNull), createdAt (timestamp withTimezone defaultNow notNull).
Note: Use `uniqueIndex` on studentId to enforce 1:1.

**Step 12 — Table 10: ai_insights**
Columns: id (uuid PK defaultRandom), studentId (uuid references students.id onDelete cascade notNull — nullable if trend insight), insightType (insightTypeEnum notNull), content (jsonb notNull — structured AI output), isCurrent (boolean notNull default true), generatedAt (timestamp withTimezone defaultNow notNull), createdAt (timestamp withTimezone defaultNow notNull).
Indexes: (studentId, isCurrent).

**Critical constraints to verify:**
- `staff_profiles.id`: NO `.defaultRandom()` — this is the single most important constraint in the schema
- `graduation_plans.studentId`: uniqueIndex (enforces 1:1 with student)

- `enrollments(studentId, classId)`: uniqueIndex (prevents duplicate enrollments)
- All timestamps use `{ withTimezone: true }`
- All FK references use the array form for indexes: `(t) => [index(...).on(t.fieldName)]`
